How to convert from the floating-point number to the fixed-point number.
The input files: inputs_0.dat
		inputs_1.dat
		inputs_2.dat
		inputs_3.dat
		inputs_4.dat

The output files: inputs_0.hex
		inputs_1.hex
		inputs_2.hex
		inputs_3.hex
		inputs_4.hex

Open convert_fixed.py to change input name. (for example:  
f1 = open("./inputs_1.dat", "r")
with open("./inputs_1.hex", "wb") as f2: )

$python convert_fixed.py
